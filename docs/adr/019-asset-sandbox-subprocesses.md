# ADR-019 · Asset sandbox subprocesses

- Status: Accepted
- Date: 2026-05-18
- Phase: 0 (pre-v1.0 — contract-exempt per risk R-03)

## Summary

FBX/OBJ importers run as syscall-restricted subprocesses with no network access. The editor cannot be crashed by a malformed input file.
