# ADR-009 · Two netcode modes

- Status: Accepted
- Date: 2026-05-18
- Phase: 0 (pre-v1.0 — contract-exempt per risk R-03)

## Summary

Authoritative (predict + reconcile, default) and Rollback (deterministic lockstep). Different genres need different netcode primitives; the engine refuses to mix them within a session.
