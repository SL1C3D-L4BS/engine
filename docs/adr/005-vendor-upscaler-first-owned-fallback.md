# ADR-005 · Vendor upscaler first, owned fallback

- Status: Accepted
- Date: 2026-05-18
- Phase: 0 (pre-v1.0 — contract-exempt per risk R-03)

## Summary

DLSS 4 / FSR 4 / XeSS 2 win on latency where supported (dedicated silicon). The owned ONNX temporal upscaler is the universal fallback. The UpscalerProvider trait keeps game code unchanged across providers.
