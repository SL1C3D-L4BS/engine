# ADR-014 · Hot/cold component separation

- Status: Accepted
- Date: 2026-05-18
- Phase: 0 (pre-v1.0 — contract-exempt per risk R-03)

## Summary

Cold components sharing an archetype with hot components incur cache-line waste on every hot iteration. The memory debugger detects violations and surfaces quantified cost.
